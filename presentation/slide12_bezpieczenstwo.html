Applied safeguards

Access

  • Entra ID — login and roles (employee / Supervisor / Director)
  • JWT in API Management — every API call with identity

Secrets

  • Key Vault — keys and parameters outside code
  • Managed Identity — Azure services connect without passwords in the repository

Data

  • Encryption SQL and Blob (at rest and in transit)
  • WORM — immutable audit of decisions in archive

Oversight

  • App Insights — logs and API request tracing
  • Threshold rule in backend and database — independent of UI
