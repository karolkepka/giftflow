GiftFlow
Pokrycie scenariusza (kroki 1–5)
1 · Zgłoszenie zapotrzebowania

Forms / Power Apps → POST /requests (cel, beneficjent, kwota).

2 · Weryfikacja progu 1000 PLN

Deterministyczna ApprovalPolicy — auto/przełożony lub Dyrektor.

3 · Decyzja + automatyczny feedback

Service Bus → powiadomienie zwrotne (e-mail).

4 · Bezpieczne archiwum chmurowe

Niezmienny audyt JSON w Blob WORM + szyfrowanie.

5 · Analityka wydatków

Fabric (star schema) + Power BI: dział × miesiąc × typ.

Atrybuty jakościowe
Skalowalność

Serverless, event-driven, scale-to-zero; rośnie z wolumenem bez przeprojektowania.

Utrzymanie

IaC (Terraform), CI/CD, observability; kod OOP testowalny (jedna klasa = plik).

Bezpieczeństwo

Entra ID RBAC (Dyrektor vs pracownik), Key Vault, WORM, separacja obowiązków.

Koszty

Pay-per-use, ~0,45 zł/wniosek; dźwignie FinOps i budżety/alerty.

Dostarczone artefakty
Architektura Azure Build vs Buy ERD 3NF DFD Sekwencja Przepływ animowany Analityka BI FastAPI (OOP) Terraform + SQL

Rekomendacja: rdzeń pro-code na Azure serverless (FastAPI), 3NF dla integralności + star schema dla analityki, GenAI punktowo (copilot BI, kategoryzacja). Rozwiązanie realizuje cały scenariusz end-to-end, jest skalowalne, bezpieczne i optymalne kosztowo.
